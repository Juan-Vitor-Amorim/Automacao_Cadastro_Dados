# Automacao_Cadastro_Dados
Imagine que você tenha produtos, preços ou qualquer tipo de informação que precisa cadastrar em um sistema.  Agora imagine que você tem 500 informações, seria um tanto trabalhoso ter que fazer tudo isso de forma manual não é  mesmo? Você perderia até alguns dias de trabalho fazendo esse processo. A ideia do projeto é que você consiga entrar em um site/sistema, fazer seu login e fazer o cadastro de todos os produtos de forma automática.  Tudo isso utilizando Python + (pyautogui) para controlar o seu mouse e teclado, + (pandas) para ler a sua base de dados.
